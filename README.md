# Heart Disease Prediction System

Machine learning-based web application for predicting the likelihood of heart disease using clinical health parameters and real-time user input analysis.

## 🚀 Live Demo
[Open Live App](https://heart-disease-prediction-1jcd.onrender.com/)

## 📌 Features
- Real-time heart disease prediction
- Interactive Flask-based web interface
- Probability-based prediction output
- Clinical data preprocessing and feature engineering
- Multiple machine learning model evaluation
- Automated model selection using ROC-AUC

## 📊 Dataset Information
- Dataset Size: 918 clinical records
- Features: 11 input health parameters
- Target Variable: Heart Disease Prediction

## 🧠 Machine Learning Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- Decision Tree

## 📈 Model Performance

| Model | Accuracy | ROC-AUC |
|-------|----------|----------|
| Logistic Regression | 86.4% | 0.904 |
| SVM | 85.3% | 0.913 |
| Random Forest | 88.6% | 0.927 |
| Decision Tree | 80.4% | 0.807 |

✅ Selected Model: **Random Forest**

## ⚙️ Preprocessing Techniques
- KNN Imputation
- Categorical Encoding
- Feature Scaling
- Stratified K-Fold Cross Validation
